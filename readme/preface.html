This is an online textbook on basic optics and visual optics (that is, optics applied to the eye). But it's also an interactive textbook. Many of the figures can be manipulated by you, the reader.

Clickable Images

The simplest sort of interaction is an image that changes when you click it. An example is below. These images do not have any visual cue that they can be clicked, but you will usually find a suggestion in the text or caption that clicking the image will do something.

Interactive Images

The other sort of image is fully interactive. These images have an icon like this: next to them. If you hold or click on the icon, the parts of the image that you can move or drag are lit up. Try it on the image below

If you ever want to get back to the original state of an interactive image, just refresh the page.
